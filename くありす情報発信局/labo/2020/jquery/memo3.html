関数の作り方

function オリジナル関数名(引数1,引数2){ return 引数1, 引数2; }

スコープの種類

グローバル プログラム内のどこでも使用できる
ローカル プログラム内の限られた範囲内でしか使用できない
※PHPで学習した内容と類似部分

命名規則

  • 使用できる文字列は、半角英数字と「$」、「_」
  • 先頭文字に数字は使用できない(不可「1num」、可「num1」)
  • 大文字と小文字は区別される(「abc」、「Abc」は区別される)
  • 予約語(JavaScriptで使われている文字列または、将来使われる予定の文字列)は使用できない
